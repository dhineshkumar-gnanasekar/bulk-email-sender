
  

# bulk-email-sender

  

Send Bulk Emails at ease.

  

## Description

  

Vite Vue with Nest Framework and MongoDB

  

- Frontend - Vite Vue with Tailwind CSS

- Backed - Nest with MongoDB

  

  

## Installation

  

  

```bash

  

$  pnpm  install

  

```

  

  

## Running the app

  

  

```bash

  

# development

  

$  pnpm  run  start

  

  

# watch mode

  

$  pnpm  run  start:dev

  

  

# production mode

  

$  pnpm  run  start:prod

  

```

  

  

## Test

  

  

```bash

  

# unit tests

  

$  pnpm  run  test

  

  

# e2e tests

  

$  pnpm  run  test:e2e

  

  

# test coverage

  

$  pnpm  run  test:cov

  

```
